    1. Opening and writing to image files
    2. Displaying images
    3. Basic manipulations
        a. Statistical information
        b. Geometrical transformations
    4. Image filtering
        a. Blurring/smoothing
        b. Sharpening
        c. Denoising
        d. Mathematical morphology
    5. Feature extraction
